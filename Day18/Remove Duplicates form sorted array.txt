Que: Remove Duplicate from Sorted Array(Easy)

Link: https://leetcode.com/problems/remove-duplicates-from-sorted-array/

Solution:

class Solution {
    public int removeDuplicates(int[] nums) {
        int i = 0;
        for (int j = 1; j < nums.length; j++) {
            if (nums[i] != nums[j]) {
                i++;
                nums[i] = nums[j];
            }
        }
        return i + 1;
        
    }
}
